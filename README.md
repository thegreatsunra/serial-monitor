# serial-monitor

> Monitors a serial port and logs output to a log file

## Dependencies

- python
- pyenv
- poetry

## Usage

```shell
## Install dependencies
poetry install

## Run main script
poetry run ./src/main.py

## Lint with black
poetry run black .

## Test with pytest
poetry run pytest

## Update dependencies
poetry update
```
